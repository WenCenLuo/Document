## 对象方法

```java
abs(x)                 //返回绝对值
```

```java
ceil(x)                 //向上取整
```

```java
floor(x)				//向下取整
```

```java
max(x,y)
```

```java
min(x,y)
```

```java
pow(x,y)				//x的y次幂
```

```java
random()                //返回0-1的随机数字
```

```java
round(x)                //把数四舍五入为最接近的整数
```

```java
valueOf()               //返回Math对象的原始值
```

```java
Math.round(Math.random()*(m-n)+n)        //随机数公式
```
